# Proyecto 3 - NYSE Bull vs Bear

## Objetivo
Predecir escenarios de mercado bull/bear y validar hipotesis estadisticas sobre rendimiento en acciones del S&P 500.

## Problema
Se necesitaba una lectura clara de patrones de retorno y riesgo para apoyar decisiones de inversion con evidencia cuantitativa.

## Rol y contribucion
- Definicion de hipotesis y estructura de pruebas.
- Analisis por volatilidad, estacionalidad y comportamiento extremo.
- Sintesis de hallazgos en formato ejecutivo para toma de decision.

## Herramientas
- Python (Pandas, Numpy)
- Estadistica inferencial
- Visualizacion de datos
- SQL para consultas exploratorias y validaciones

## Resultado
En dias extremos del mercado, la proporcion de casos bull subio a 62% frente a 47% en dias normales.
