# Proyecto 4 - Deteccion de fraude digital

## Objetivo
Construir un piloto analitico para detectar fraude financiero con alta precision y buena interpretabilidad operativa.

## Problema
La organizacion necesitaba reducir perdidas por fraude sin aumentar falsos positivos que afectaran la experiencia del cliente.

## Rol y contribucion
- Entrenamiento y comparacion de modelos supervisados (arbol podado y SVM).
- Analisis no supervisado con K-means/PAM para segmentacion de riesgo.
- Recomendacion final basada en equilibrio entre metricas e interpretabilidad.

## Herramientas
- Python
- scikit-learn
- SQL para preparacion y validacion de datos
- Arboles de decision, SVM, K-means

## Resultado
El arbol podado mostro desempeno muy alto en test y se priorizo por interpretabilidad para despliegue controlado.
